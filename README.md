# TikTok Matrix Analytics Dashboard

A high-performance CSV processing tool designed to transform TikTok content IDs into clickable direct links while maintaining full data integrity and providing a clean, professional dashboard view.

## Features

- **CSV Parsing**: Local processing of large datasets without uploading to external servers.
- **Matrix View**: Switch between Product, Creator, and Video specific views.
- **Deep Linking**: Direct links to TikTok videos and creator profiles.
- **Export**: Generate clean, UTF-8 BOM formatted CSV reports for Excel.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`

3. Build for production:
   `npm run build`
